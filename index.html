map SNS (webアプリ)

使用技術 ： html, css, javascript, React, firebase, firestore, google map API

URL ： https://map-ap-bc383.web.app

github ： https://github.com/kwsmyy/map-ap

概要 ：
マップ上で繋がるSNSを作成しています。（現在、試作段階ではありますが、数名ユーザがいます。）
マップ上で位置情報とテキストを投稿し、ユーザ同士で相互に公開・閲覧ができます。
また、フォロー・フォロワー機能やタイムライン機能も実装されています。

PC,スマートフォンの両方で使えるようにデザインを調整しています。
ユーザ登録はどなたでもできるので是非使ってみてください！

フロントエンドはReact,バックエンドはfirebase,DBはfirestoreで作成しています。
現在React NativeでiOSアプリを作成中です。
